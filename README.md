# SuperhostOS

The Operating System for Modern Hosts.

AI-assisted guest communications and operational workflows for hospitality teams.

## Stack
- Next.js
- TailwindCSS
- Framer Motion
- Vercel

## MVP
- Luxury landing page
- Interactive AI guest demo
- Operations dashboard
- Task creation workflow

## Deploy
```bash
npm install
npm run dev
vercel --prod
```
